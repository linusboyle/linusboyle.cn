## 作为一个主题：

我在持续不断地增强我的博客，也许有一天它会成为新的主题，到目前为止，它有如下新特性：
- 一些细节的优化
- 使用jekyll-feed 自动生成atom feed
- 使用jekyll-sitemap 自动生成网站地图
- 增加更多的社交网站入口
- MathJax支持 可以在博客中插入数学公式了
- 移除已经废弃的多说评论支持，加入gitalk支持
- 移植google analytics
- 增添归档
- 增添标签索引页
- 增添404页面
